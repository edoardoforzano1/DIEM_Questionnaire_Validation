# Data Sources Overview

Validation combines multiple data sources depending on mode and tool.

| Source | Used for | Tool |
|---|---|---|
| Current questionnaire | Primary object under validation | KoBo + GeoPoll |
| Latest template | Canonical baseline and placeholders | KoBo + GeoPoll |
| Previous round file | Continuity baseline | KoBo + GeoPoll |
| Additional information sheet | Placeholder value replacements | KoBo + GeoPoll |
| Crop list sheet | Country crop options and checks | KoBo + GeoPoll |
| AGOL admin boundaries | Admin1/Admin2 refresh (where applicable) | KoBo |

See dedicated pages for input structure and expectations.
